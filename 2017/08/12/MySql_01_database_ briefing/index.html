MySql_01 数据库简介

Aug 12, 2017 | MySql

MySql 也就是数据库，那么：

  • 什么是数据库？
  • 什么是数据库系统？
  • 什么是数据库管理系统？

1、什么是数据库？

数据库(database)就是存储数据的仓库，为了方便数据的存储和管理，将数据按照特定的规律存储在磁盘上，通过数据库管理系统，有效的组织和管理存储在数据库中的数据。

2、什么是数据库系统？

数据库系统和数据库不是一个概念，数据库系统(DBS)，比数据库大很多，有数据库、数据库管理系统、应用开发工具构成。

3、什么是数据库管理系统？

数据库管理系统(DataBase Management System,简称DBMS)，用来定义数据、管理和维护数据的软件。他是数据库系统的一种重要的组成部分。

常见的数据库系统：
  • 1、甲骨文Oracle数据库
  • 2、IBM的DB2
  • 3、微软的SQL Server、Access
  • 4、PostgreSql
  • 5、MySQL

MySQL 的优点

  • 开放源代码
  • 跨平台
  • 免费（并非不要钱，如需技术支持需要付费，不过很低）
  • 功能强大使用方便

SQL简介

1、什么是SQL？

Structured Query Language 简称SQL，结构化查询语言，是一种数据库查询和程序设计语言，用于存取数据以及查询、更新和管理关系数据库系统；同时也是数据库脚本文件的扩展名。

2、SQL语言的组成部分？

  • DDL(Data Definition Language)：数据定义语言，主要用于定义数据库、表、视图、索引和触发器等
  • DML(Data Manipulation Language)：主要包括对数据的增删改查。
  • DQL(Data Query Language)：数据库检索语句，用来获得数据，确定数据怎样在应用中给出。
  • DCL(Data Control Language)：数据控制语言，主要用于控制用户的访问权限。
