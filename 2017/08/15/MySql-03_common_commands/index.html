常用命令

  • SELECT VERSION();　　 显示当前版本
  • SELECT NOW();　　　　 显示当前日期时间
  • SELECT USER();　　　　显示当前用户

MySQL语句的规范

  • 关键字与函数名称全部大写
  • 数据库名称、表名称、字段名称等全部小写
  • SQL语句必须以分隔符结尾
  • SQL语句支持折行操作，只有不把单词、标记或引号字符串分割
  • 数据库名称、表名称、字段名称不要使用MqSQL保留字，如果需要，必须用反引号（‘’）括起来。
