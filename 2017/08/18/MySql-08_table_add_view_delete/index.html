Aug 18, 2017 | MySql

1、什么是表？

  • 数据表是数据库最重要的组成部分之一，是其他对象的基础。
  • 数据表示存储数据的数据结构。
  • 数据表包含了特定·试题类型烦人的数据。
  • 数据表有行(row)和列(column)构成烦人二维网络。
  • 数据表一定现有表结构，再有数据。
  • 数据表至少有一列，可以没有行或者多行。
  • 数据表名称要求唯一，而且不要包含特殊字符。
