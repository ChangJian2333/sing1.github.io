MySql_13 MySQL中的子查询

Sep 19, 2017 | MySql

1、什么是子查询？

子查询是将一个查询语句嵌套在另一个查询语句中。内层查询语句的查询结果，可以作为外层查询语句提供条件。

2、引发子查询的情况？

  • 使用[NOT]IN的子查询
  • 使用比较运算符的子查询（=、>、 <、>=、 <=、 <>、 !=、 <=>）
  • 使用[NOT]EXISTS的子查询
  • 使用ANY|SOME或者ALL的子查询
运算符 ANY SOME ALL
>、>= 最小值 最小值 最大值
<、<= 最大值 最大值 最小值
= 任意值 任意值
<>、!= 任意值
