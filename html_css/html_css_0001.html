环境搭建

一个好的开发工具对开发效率来说事半功倍，这里选择里Atom。

何为ATOM

Atom是专门为程序员推出的一个跨平台文本编辑器。具有简洁和直观的图形用户界面，并有很多有趣的特点：支持CSS，HTML，JavaScript等网页编程语言。它支持宏，自动完成分屏功能，集成了文件管理器。

为什么用ATOM

  • 开源 — 遵循MIT协议,代码托管在github上
  • 多平台 — 支持MAC/WIN/LINUX（支持源码编译安装,也提供二进制安装包）
  • 丰富的插件库 — 各种插件丰富,且支持在线更新
  • 类Sublime — 风格和sublime text极其相似,风格、操作和快捷键等都是通用的
  • 采用包管理技术 — 采用了node.js来访问文件系统和包管理。
  • 强大的生命力 — 背靠github社区，可以注入源源不断的生命力
