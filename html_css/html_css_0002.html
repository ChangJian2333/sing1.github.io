HTML的语法

1、什么是HTML标记语言

HTML标记语言表示网页信息的符号标记语言。特点如下：

  • 可以设置文本的格式，比如标题、字号、文本颜色、段落等等
  • 可以创建列表
  • 可以插入图像和媒体
  • 可以建立表格
  • 超链接，可以使用鼠标点击超链接来实现页面之间的跳转

6、字符实体

什么是字符实体？

比如我们想在网页上面显示一个“<”小于符号，但是“<”在HTML中是文档标记的开始语言，如果我们直接使用“<”会出差错，所以我们就会一些实体名称来代替！
