1、数值型
  • 十进制数

  • 八进制数

  • 特殊值（Infinity，NaN）

  • 注：1、Infinity为无穷大（1.79e309，-1.79e309）
    　　2、NaN，不是一个数字类型，比如：0/0，可以通过isNaN()检测值是否为NaN，不可以与自身比较（if(NaN == NaN)）

2、复合数据类型

  • 对象(object)
  • 数组(array)
  • 函数(function)

3、特殊数据类型

  • 无定义数据类型（undefined）
  • 空值（null）

1、undefined，未赋值或不存在的值
2、null，代表为空值，与undefined不同

4、变量

声明变量

  • 通过var关键字声明变量
  • 可以声明变量的同时给变量赋值

注意：

  • 未赋值的变量，默认值为undefined
  • 如果变量重名产生覆盖
  • 变量严格区分大小写
  • 变量名称不要包含特殊字符
  • 变量名称最好遵循驼峰标记法或者下划线法
  • 变量名称最好含义明确
